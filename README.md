# Lootbox
A lootbox plugin that let's you create as many type of lootboxes as you want with 5 animation choices.
 - Supports PiggyCustomEnchantments

# Required virions
 - InvMenu

# Commands
| Command | Description |
| --- | --- |
| `/givelootbox <player> <identifier> [amount = 1]` | Give a loot box to a player |
